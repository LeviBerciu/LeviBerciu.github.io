Memory Game

MEMO CUBE


Memo Cube is my take on the classic "Match'Em" game.


I got the idea for this project while I was learning game development from the “Unity in Action” book. It was at “Chapter 5: Building a Memory game using Unity’s 2D functionality” when it occurred to me, this could also work in 3D.


The design focuses on geometric shapes and bold colors while maintaining a truly minimalist aesthetic.
